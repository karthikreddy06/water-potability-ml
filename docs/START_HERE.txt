================================================================================
📈 MODEL EVALUATION METRICS
================================================================================

ACCURACY
├─ Definition: Overall correctness %
├─ Formula: (TP + TN) / Total
└─ Use: When classes are balanced

PRECISION
├─ Definition: Of predicted safe, how many actually are?
├─ Formula: TP / (TP + FP)
└─ Use: When false positives are costly

RECALL
├─ Definition: How many actual safe samples did we find?
├─ Formula: TP / (TP + FN)
└─ Use: When false negatives are costly

F1 SCORE (PRIMARY SELECTION METRIC)
├─ Definition: Balanced combination of precision & recall
├─ Formula: 2 × (Precision × Recall) / (Precision + Recall)
└─ Use: To select best model

CONFUSION MATRIX
├─ TP (True Positive)      Correctly predicted potable
├─ FN (False Negative)     Predicted non-potable, actually potable
├─ FP (False Positive)     Predicted potable, actually non-potable
└─ TN (True Negative)      Correctly predicted non-potable
